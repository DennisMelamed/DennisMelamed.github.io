Micro-UAV Door Flight (In Progress)

The aim of this project is to have a Crazyflie micro-UAV (~50g) equipped with a low-resolution camera safely exit a room through the door from a random initial location.

In order to successfully fly through a door, the door must first be detected in the noisy, low-resolution onboard camera image. Two methods are explored, one based on image geometry, the second using a fully convolutional neural network.

Image Geometry

The Hough transform is used to determine possible lines (i.e. doorframe edges) in the image. Due to noise, spurious detections are common and are filtered using a clustering technique. The remaining lines are used to determine possible vanishing points for the doorframe plane in the image, and the image is then rectified to be fronto-parallel to the camera using those vanishing points. Rectangles with a door-like aspect ratio are found in the rectified image. Due to noise, rectification is not always successful so the rectangles found in good rectifications are used to initialize tracking of the likely door in the original image stream.

Micro-UAV Outdoor Navigation (In Progress)

A combination of segmentation by color and the vanishing point theorem is used to select a good point to fly toward in order to exit a corn row.

The optical flow sensor available with the Crazyflie is used to navigate around an apple.

A Gesture Based Programming Scheme for the Crazyflie Micro-UAV

A ROS package allowing for gesture control of a micro-UAV using a Leap Motion controller. Gestures are compiled and saved into macros, which can then be run or called recursively from other macros.
